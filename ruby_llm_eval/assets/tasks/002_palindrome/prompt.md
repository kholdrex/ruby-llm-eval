# Palindrome check

Implement a method with this exact signature:

```ruby
def palindrome?(str)
end
```

Return `true` if `str` is a palindrome, `false` otherwise. Comparison must:

- ignore letter case, and
- ignore every non-alphanumeric character (spaces, punctuation, etc.).

The empty string is a palindrome.

Examples:

```ruby
palindrome?("A man, a plan, a canal: Panama") # => true
palindrome?("hello")                          # => false
```
