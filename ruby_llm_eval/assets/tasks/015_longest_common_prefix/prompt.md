# Longest common prefix

Implement a method with this exact signature:

```ruby
def longest_common_prefix(strings)
end
```

Return the longest string that is a prefix of every string in `strings`.
Return `""` if there is no common prefix or if `strings` is empty.

Examples:

```ruby
longest_common_prefix(["flower", "flow", "flight"]) # => "fl"
longest_common_prefix(["dog", "cat"])               # => ""
longest_common_prefix([])                           # => ""
```
